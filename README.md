# FinPath AI – Smart Personal Finance Advisor

FinPath AI is a web-based personal finance advisory platform developed using Python and Flask. It helps users evaluate financial health, calculate SIP requirements, plan financial goals, analyze risk profiles, and receive personalized investment recommendations.

## Features

- Risk Profiling
- Financial Health Score
- Financial Health Rating
- Goal Planning
- SIP Calculator
- Asset Allocation Recommendations
- Pie Chart Visualization
- Emergency Fund Analysis
- Finance Assistant Chatbot

## Technologies Used

- Python
- Flask
- Bootstrap
- HTML/CSS
- JSON
- Matplotlib

```bash
pip install -r requirements.txt
python app.py
```
